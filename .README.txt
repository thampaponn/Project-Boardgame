***Open mainmenu.py and run(F5) to play the game***
- Rock Paper Scissors : กดรูปค้อน กระดาษ หรือกรรไกร เพื่อเป็นตัวเลือกของเรา จะมีแถบคะแนนบอกด้านล่างว่าชนะ แพ้ เสมอ กี่ครั้ง
- Fast Typing : คำจะค่อยๆตกลงมาจากข้างบน หากพิมพ์ตัวใดผิด จะต้องพิมพ์ทั้งคำใหม่ จะมีแถบแสดงด้านล่าง
- Tic Tac Toe : สำหรับเล่น 2 คน
- Snake Game : เล่นเหมือนเกมงู แต่หากเรากำลังไปทิศใดทิศหนึ่ง แล้วกดไปทิศตรงกันข้าม เช่น กำลังไปทางขวา แล้วกดลูกศรซ้าย จะตายทันที
- Hangman : เล่นปกติ มีโอกาส 6 ครั้ง
